Quotations

On this web page, we will go over how to use quotes. You can site blockquotes by using the tag <blockquote cite="website.com">

You can also do short quotes with <q>, Which is a way of typing "" without actually typing the quotes.
When using a quotation, you must set the style of the quote separatly from any tags that the quotation is imbedded in

Addresses

Addresses <address> give the text within the tags a nice italicized font to make used to define contact information.
For Example:

Ben Podrazhansky
View Github at: https://github.com/BPod123
Email@address.com

Citing information

The <cite> tag works like the <i> tag.

Bi-Directional Overide <bdo dir="direction">

The Bi-Directional Override lets you set the direction of the text. For example, you can have everything in the tag appear in reverse order with <bdo dir="rtl">

Abbreviations


 Use <abbr title=name> A.B.B.R.E.V.I.A.T.I.O.N. </abbr> To abbreviate something like the CIA.

Hovering your mouse over it tells you what it stands for.
