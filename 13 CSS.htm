CSS

    Types of CSS

  • Inline
  • Internal
  • External

Inline CSS

Inline CSS is just using the syle attribute as an attribute of an element.
<'p style="text-align:center"'><'/p'>

Internal CSS

Internal CSS uses a style tag in the head and defines style for element types as oppose to individual instances of an element with inline CSS.

        To set all h3 sized heading to have a yellow background, have the following in your head:
        <head>
            <style>
                h3	{background-color:yellow;}
            </style>
        </head>

External CSS

External css is where in the <head>, you link to a .css file with all of the stylings. It is like Internal CSS, where it defines styles for element types, but the difference is that it is retrieved from a .css file. An advantage to this can be if you like a particular format for all of your pages and you don't want to have to define the format in the <head> of every page. 

        Use it like this:
        
        <head>
            <link rel="stylesheets" href="path/to/styles.css">
        </head>

CSS Fonts

    There are multiple ways to manipulate text fonts with CSS

  • Color: color:blue
  • Font: font:family:cursive
  • Font size in pixel height: font-size:30px
  • Font size in percentage: font-size:150%
  • Border of element: "border: 5px solid yellow"
  • Padding with percentage or pixels: padding:1px or padding:20%
  • margin with percentage or pixels: margin:10px or margin:10%

id Attribute

You can give elements there own id in the tag. This can be usefull if you want to define its stile using internal CSS by defining it in the <head>.

        <head>
            <style>
                #p1	{
                                background-color:blue;
                            border: 3px solid red;
                                    color:white;
                    }
            </style>
        </head>
        <body>
            <p> This is a normal paragraph </p>
	<p id=p1> This paragraph has the style set in the head.</p>

class Attribute

The class attribute is similar to an id, but an id should be unique, and a class should be used multiple times. Lets have the class "log" and I want any paragraph that is a log to be in blue text.

         <head>
            <style>
                p.log	{color:blue;}
            </style>
        </head>
        <body>

This is a normal paragraph

<p class = log > This paragraph is a log </p>
