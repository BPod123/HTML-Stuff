Minimum requirements for an image: The src Attribute

An image tag must have the keyword img and the image file or url to image
<img src="filename.jpg">
Or:
<img src="https:urlOfImage.com">

The alt Atribute

The alt attribute lets you set text to be displayed when the image cannot be found.

Image tag: <img src="path/to/nonexistant/file.jpg" alt = "Picture of the most awesome thing ever.">

What is seen by the user:

You can use the width and height attributes to set the size of the image

<img src="url or file" width = "(int)width in pixels" height = "(int) height in pixels" >

Be mindful that images have a certain aspect ratio and messing with the width and height, if not done properly, can distort the image.

Set Attributes with style in head 

<head>
	<style>
    	img {	width:100%;
        		height:50;
			}
    </style>
</head>
