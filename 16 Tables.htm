Defining Tables

Define a table with the <table>. Table rows are defined with <tr> Within a row are table headers <th> and table data cells <td>. Table headers <th> are bolded and centered while table data cells are normal.

Code for Table 1:

    <table>
        <tr>
        <th> Table header 1 </th>
        <th> Table header 2 </th>
        </tr>
        <tr>
        <td> Row 2 data 1 </td>
        <td> Row 2 data 2 </td>
        </tr>
    </table>

Adding borders

You can add borders to your tables by using CSS.
style="border:1px solid white", or by setting the style for tables, rows, headers, and data cells in the head. You can also set the table width to 100% to fill out the page as follows:
    <head>
        <style>
            table, th, td	{border:1px solid white;}
        </style>
    </head>
    
    Then add a table normally:
    
    <table style="width:100%;">
        <tr>
        <th> Table header 1 </th>
        <th> Table header 2 </th>
        </tr>
        <tr>
        <td> Row 2 data 1 </td>
        <td> Row 2 data 2 </td>
        </tr>
    </table>

 Making a single border

    
 Have the border be a single border; one shared between table, rows, headers, and data cells; with style="border-collapse:collapse"

    <head>
        <style>
            table, th, td	{border:1px solid white;
                            border-collapse:collapse;}
        </style>
    </head>
