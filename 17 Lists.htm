Lists

Ordered and Unordered Lists

There are two types of lists. Ordered lists <ol> have numbered list items <li>, while unordered lists <ul> have list items <li> shown in bullet points.

                
                
 Ordered List:

    <ol>
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
    </ol>
                
    
                    
  1.  Item 1 
    2.  
                    
  2.  Item 2 
    3. 
                    
  3.  Item 3 
    4. 
                

                
                
 Unordered List:

    <ul>
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
    </ul>
                
    
                    
  •  Item 1 
    •  
                    
  •  Item 2 
    • 
                    
  •  Item 3 
    •

Unordered List Customization:

Different types of bullet points can be chosen with CSS in the list tag <ul>.


        
 Disc Points: 
 
        
    <ul style ="list-style-type:disc;">
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
    </ul>
    
    
        
  •  Item 1 
    •  
        
  •  Item 2 
    • 
        
  •  Item 3 
    • 
    

    

    
 Circle Points: 

    
    <ul style ="list-style-type:circle;">
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
    </ul>
    
    
        
  •  Item 1 
    •  
        
  •  Item 2 
    • 
        
  •  Item 3
