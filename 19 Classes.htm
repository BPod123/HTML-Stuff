Use classes to define the style for elements of that class.

In the tag for an element, give it the 'class="className"' attribute to assign it to a class which can be defined in the header or in a .css file.
Any type of HTML element can belong to a class. 

            <head>
            <style>
            /* Define a class name with ".classname"
            The follwing example will be of a class we will call cringe, because 
            instances of it will make you cringe.*/
            .cringeyClass    {
                /** Edit the class as you would anything else with CSS */
                font-family: cursive; 
                background-color: yellow;
                color: aqua;
            }
            </style>
        </head>
        <body>
            <div class="cringeyClass">
                <h3> Here is a header. </h3>
                <p> And here is a paragraph. </p>
                <p> I am disapointed in myself for making this.</p>
        </div>
        </body>
