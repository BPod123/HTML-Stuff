The 'id' Attribute

The id attribute is used as a sort of label to use with CSS and JavaScript. You can assign an id to one or multiple elements and any CSS or JavaScript that manipulates elements based on their id can affect elements with that id.
Note that using the same id for multiple elements will work with CSS, but not JavaScript

Using id with CSS

The following code uses CSS to manipulate any elements with the id "cssEdit" by defining the style for elements with that id in the header.

        <head>
            <style>
                #cssEdit    {background-color:red;
                        color:white;
                        text-decoration: underline;}
            </style>
        </head>
        <body>
            <h3 id = "cssEdit"> This header has the id "cssEdit" </h3>
            <p id="cssEdit"> This paragraph also has the id "cssEdit", however this is not good practice. </p>
                defined in the <head>
            </p>
        </body>

This header has the id "cssEdit"

This paragraph also has the id "cssEdit", however this is not good practice.

Using id with JavaScript

Consider the following code and its following implementation:
            <p id="target"> This paragraph will be changed when a button is pressed. </p>
            
            <!-- Button to change text -->
            <button type="button"
            onClick='document.getElementById("target").innerHTML = "This paragraph has been edited with JavaScirpt."'>
            Change Text </button>
            
            <!-- Button to revert text back -->
            <button type="button"
            onClick='document.getElementById("target").innerHTML = "This paragraph will be changed when a button is pressed."'>
            Change text back</button>

This header will be changed when a button is pressed.

This paragraph will be changed when a button is pressed.

Here we can see a problem with using multiple id's. JavaScript will only find the first element with the specified id.
The paragraph will not be changed even though it has the same id as the header.

When to use class or id

It is good practive to use class when applying it to multiple elements and id with individual elements.
Both classes and id's can be used for all html elements.
