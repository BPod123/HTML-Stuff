Iframes <iframe>

An <iframe> is a window within a window (window-ception). It is defined as follows:
<iframe src=URL> </iframe>
You can also apply width, and height attributes to an iframe as you would an image <img>

<iframe src=url width=500 height=500></iframe>

            <iframe src=20 Id.htm width=500 height=500></iframe>

CSS with Iframes

            <iframe src=13 CSS.htm style="color:black; background-color:white; width:500px; height:500px;"></iframe>

Note that if the iframe is inside of an element that has had its style set with CSS, the style may carry over into the iframe unless it is undone in the style of the iframe.

Make links be displayed in iframes

You can set the target of a link to be an iframe. All you have to do is give the iframe a name with the name attribute. 

            <iframe src=https://github.com/BPod123/HTML-Tutorial width=500 height=500 name = "linkTarget"></iframe>
            <a href="14 Links.htm" target="linkTarget"> Tutorial on Links </a>
