Defining what a file is

A file path is exactly what it sounds like; a path to a file. As you probably know, computers have a file system that organizes files. Every file has a unique location that it does not share with any other file.

When we say the word file in everyday use, we are refering to one of two types of files:
Directories
Files that hold other files
Files
Files with content, such as a text or word document.

Absolute and relative file paths

An absolute file path is location of the file on the computer for example:
"C:\Users\ben\Documents\example.txt"
A relative file path is the location of one file relative to the location of another. A relative file path will not often not be a top down path as shown above. Sometimes you have to go up then over.
