What is the <head>

In short, <head> stores metadata. Metadata is data about other data. The <head> can be used to define CSS> stylings for multiple elements at once, link to css file that define stylings, set the title of the tab, and more.

The <title> tag

The <title> is the title of the page. You can use it to set the text that is displayed in the tab. By default, when no <title> is present in the head, the title will be the name of the file.

The <style>

The <style> tag is used to define CSS> for the page. Within the <style> tag, code is writtin in CSS>, not HTML. When you see the <style> tag, think CSS> file, because you are essentially making one for the document.

The <script> tag

The <script> tag is like the <style> tag except instead of CSS>, the code within is in JavaScript.

The <link> tag

The <link> tag is used to link other files to this one. You can link a CSS> file to with the following format:

<link rel="stylesheet" href="url/or/path/to/file.css">

You can use CSS> to set the style for multiple pages. It is more efficient than using the same <style> tag for every page.

The <meta> tag

The <meta> tag is used to store information about the page that is not necessarily seen on the page. You can use it to store the author of page's name, a description of the page, keywords associated with the page, the refresh rate, and more. The format is as follows:

<meta name="name of item" content="information about item">

Examples:

<meta name = "author" content="Ben Podrazhansky">
The author of the page is set to Ben Podrazhansky now.
<meta name="description" content="Awesome HTML Tutorial">
Now there is a descritption for the page.
<meta name="keywords" content="HTML, CSS, JavaScript">
The content of the page is now associated with "HTML" in searches.
<meta http-equiv="refresh" content="30">
This one is a little different. It sets the refresh rate of the page to once every 30 seconds.
<meta charset="UTF-8"
Set the character set of the page.

The <base> tag

The <base > tag is used to set the starting point in the file system for all file pahts in links, images, and anything else that uses a file system.
<base href="url/or/file">
