What is the <head>

In short, <head> stores metadata. Metadata is data about other data. The <head> can be used to define CSS stylings for multiple elements at once, link to css file that define stylings, set the title of the tab, and more.

The <title> element

The <title> is the title of the page. By default, when no <title> is present in the head, the title will be the name of the file. With the <title> you can set the text that is displayed in the tab.
