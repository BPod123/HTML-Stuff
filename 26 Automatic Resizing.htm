Automatic Image Sizing With the auto keyword

When setting the width and height of an object, you can set one parameter to be based on the other with auto. The width and height of an object are both set to auto by default.
For Example: 

                        <img src="images/mountain.jpeg" width=50%; height=auto>
                        or use CSS>
                        <img src="images/mountain.jpeg" style="width:50%; height:auto">
                Try resizing the window. The pictues width and height should change.

max-width

The max-width property is used to set the maximum width that an image/element can be. Note that it doesn't set the width, it only limits the maximum it can be.

The <picture> tag

When you want to have different images for different sized windows, use a <picture> tag.
Resize the window to change the image. 

                    <picture>
                        <source srcset="images/smallMountain.jpeg" width=250px media="(max-width: 600px)">
                        <source srcset="images/mountain.jpeg" width=300px media="(max-width:800px)">
                        <source srcset="images/bigMountain.jpeg">
                        <img src="images/bigMountain.jpeg">
                    </picture>

Change font size with view width

You can set the font size of text with view width in CSS as follows:
style="font-size:Xvw"
where x is a number.

<h3> Header 3 </h3>

<h3 style="font-size:4.8;"> Header 3 </h3>

<h3 style="font-size:1vw;"> Header 3 </h3>

<h3 style="font-size:7vw;"> Header 3 </h3>

Note that their are actually 3 ways to set font-size of text:

% px vw
Description: Sets font size based on percentage of default text size Sets font size to a specifix height in pixels Sets font size based on size of window and will change with the size of the window
Example:
Default:
<p> Normal size paragraph </p>

Normal size paragraph

 <p style="font-size:50%;"> 50% size of normal paragraph </p>

50% size of normal paragraph

<p style="font-size:30px;"> 30 pixel tall text </p>

30 pixel tall text

<p style="font-size:6.8vw;"> 6.8 view width </p>

6.8 view width. Try resizing window.

You can use media queries to put elements next to each other when the screen is a specific size. Make the elements that you want side by side to take up different widths that are percentages of the screen width and have them float left. 

    <head>
        <style>
            .thirty {
                float: left;
                    width: 30%; /* Default width value */
                    }
            .forty {
                float: left;
                    width: 40%;
                    /* Default width value */
                    }
                    /* The sections will only be in a row is the as wide or wider than 600 pixels 
                        The thirty, forty, and other thirty will have their widths add up to 100% of the screen*/
                    @media screen and (max-width: 600px) {
                        .thirty,
                        .forty,
                        .thirty {                                
                            width: 100%
                            }
                        }
        </style>
    </head>
    <body>
        <div class = "thirty" style="background-color:lightblue; color:black;">
            <h3> Section 1 </h3>
        </div>
        <div class = "forty" style="background-color:brown;">
            <h3> Section 2 </h3>
        </div>
        <div class = "thirty" style="background-color:green;">
            <h3> Section 3 </h3>
        </div>
    </body>
