The src Attribute

The src Attribute is a file name.

The alt Attribute

The alt Attribute is text that will be displayed if unable to display
an image.

The title Attribute

The title attribute is used in the paragraph tag like style. When you hover your mouse over the text, it displays the title.

The style Attribute

The stype attribute goes in the tag for the text.
For Example:

<'p style=color:red'> "Red Text here <'/p'>

The width and height Attributes

The width and height Attributes are used to set the size of an image.
It will look somthing like this:
<'img src="filename" width="number of pixels" height="number of pixels"'>

The lang Attribute

The lang attribute declares the language that the page is in. It goes in the html tag as follows <' html lang="en-US"'>
