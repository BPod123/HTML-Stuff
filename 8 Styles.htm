Styles

<'tagname style="property:value; property:value"'>

Color

color:red<\p>

Font

Make anyone who views your web page hate you for doing everything in comic sans for the rest of this page.
<'style="font-family:cursive"'>

Size

Set font size in pixels "font-size:(n pixels)px"

Or set size as percentage of original size with "font-size=200%"

Background

Background works like color, but for the body tag.
<'body style="background-color:powderblue"'>

Text Alignment

Set the text alignment with <'style="text-align:center"'>
